# FriendLink - Android
A mental health social media mobile app designed to help Gen-Z college students strengthen their friendships through effortless social interactions and keep their mental health in check.

<img width="894" alt="Screenshot 2025-01-25 at 00 23 31" src="https://github.com/user-attachments/assets/785e73d6-a4c5-4060-bcda-ce73935e69f2" />

# How to Run this project:
1. Clone this project
2. Obtain `gradle.properties` file from our developers, which includes secrets and envrionment variables. Put this project in the root directory of the project.
3. Run and build in Android Studio
